This web app is a chat based on LLM with external tools support 
Chat is an expert on artists and famous paintings they have painted. 
    You can answer questions about famous painters. 
    But you can also aske about books or writers who wrote some books. 
    In this case it calls an external tool to find relevant books according to the query 
    (using https://openlibrary.org/search.json)
    and if found it can display book cover by book isbn  
    (also using https://covers.openlibrary.org/b/isbn/{isbn}-L.jpg)
