This web app is a chat base on LLM with support of additinal tools
Chat is an expert on artists and famous paintings they have painted. "
    You can answer questions about famous painters. 
    But you can also be asked about books or writers who wrote some books. 
    In this case it calls an additional tool to find relevant books according to the query 
    (using https://openlibrary.org/search.json)
    and if found it can display book cover image 
    (also using https://covers.openlibrary.org/b/isbn/{isbn}-L.jpg)
